# Weather Detector V2.0 🌤️

A modern, feature-rich weather application built with Vanilla JavaScript. Get real-time weather data, 5-day forecasts, and a beautiful UI with dark mode support.

**Live Demo:** [your-netlify-link-here] 

![Weather App Screenshot](screenshot.png)

## ✨ Features

- **Real-Time Weather Data** - Current temperature, humidity, wind speed, and weather conditions for any city worldwide
- **5-Day Forecast** - Extended weather prediction with daily temperature and conditions
- **Geolocation Support** - Get weather for your current location with one click 📍
- **Dark/Light Mode** - Toggle between themes for comfortable viewing 🌙☀️
- **Search History** - Quick access to your last 3 searched cities using LocalStorage
- **Dynamic Backgrounds** - UI background changes based on current weather conditions
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Error Handling** - User-friendly messages for invalid cities or API failures

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **API:** OpenWeatherMap API for weather data
- **Storage:** LocalStorage API for search history and theme preference
- **Deployment:** Netlify

## 🚀 Key Concepts Used

- REST API Integration using `fetch()` and `async/await`
- DOM Manipulation and Dynamic Content Rendering
- Geolocation API for user location
- CSS Variables for theming
- LocalStorage for data persistence
- Error Handling and Loading States
- Responsive CSS Grid and Flexbox
